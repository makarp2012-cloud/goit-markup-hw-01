# Goit-markup-hw-01

A clean and semantic HTML-only markup for the "WebStudio" landing page. This repository contains the first practical homework layout, focused on using proper structural tags without CSS styling.

## 🛠️ Features Included:
- **Semantic Structure:** Implemented using `<header>`, `<main>`, `<section>`, and `<footer>`.
- **Navigation & Contacts:** Formatted inside header and footer lists.
- **Image Optimization:** Correctly linked assets for the team members section inside the `images/` folder.
- **Accessible Elements:** Proper button types and heading hierarchies (`<h1>`-`<h3>`).

## 📁 Project Structure:
- `index.html` — main page markup.
- `images/` — contains team member photos (`team-1.jpg` to `team-4.jpg`).
- `.gitignore` — specifies untracked files for Git.
